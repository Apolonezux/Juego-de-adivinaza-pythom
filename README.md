# Juego-de-adivinaza-python
Este es un juego sobre adivinar un número del 1 al 10 en python como proyecto básico, todo esto gracias al esfuerzo de un mes junto a la documentacion de freecodecamp https://www.freecodecamp.org. Espero disfrutes este juego.
## Cómo instalar
Para instalar este juego ocupas descargar el archivo .py y una vez instalado debes de ejecutarlo en una terminal, en ```linux``` es: python3 "archivo.py"
## ¿Cómo jugar?
En la terminal debes de poner un número del 1 al 10 hasta que adivines el número aleatorio.
